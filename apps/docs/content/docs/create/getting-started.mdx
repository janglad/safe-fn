## Installation

<Callout type="info">
  Please note that due to an incompatibility in types, NeverThrow version 8.1.0
  or higher is required.
</Callout>

<Tabs items={["npm", "pnpm", "yarn", "bun"]}>
{/* <!-- prettier-ignore --> */}
  <Tab value="npm">
  ```bash 
  npm i safe-fn zod neverthrow
  ```
  </Tab>
  
  <Tab value="pnpm">

  </Tab>

## Instantiating

You can create a SafeFn by using the `createSafeFn()` method.

```ts
import { createSafeFn } from "safe-fn";

const myySafeFunction = createSafeFn();
```
